Environment

Take a look at the article Installing & Using the Dev Environment to learn how to install the basic environment with all the requirements and the basic settings.

Tools

  • Firefox 23.0.1
  • Git
  • Geany / Notepad++

Frameworks & Libraries

Server Side

  • Node.js: JS Platform for building fast, scalable network applications
  • MongoDB: Document NoSQL database, for persistency
  • Node.js modules:
    • express: Web dev framework to get a nice MVC architecture
    • ejs / ejs-locals: Template engine for the views
    • socket.io: Websockets made easy, for the Server/Clients dynamic communications
    • mongoose: To interface with MongoDB
    • connect: To deal with session, to give access to identified users only
    • bcrypt: Encrypting the users passwords
    • session.socket.io: To secure the Websockets communications using the created sessions

Client Side

  • JQuery: Web dev swiss knife
  • Modernizr: Library to deal with browsers support
  • google-diff-match-patch: Library use to identify transformations done to a text input
  • JQuery plugins:
    • jquery.event.drag: Drag'n drop plugin
    • jquery.input: To get events for any kind of input
